# prompt

1. 数据源：tushare，我有权限获取数据；

2. 个股基础信息：个股名称、个股代码、当日涨跌幅、总市值、市盈率、所在行业、所属概念、所在板块、交易所、成交量、成交额；

3. 期望的数据模型结构需要包含

    1. 基础数据：对个股进行分析所需要的基础数据源，也是从 API 中获取到基础数据；

    2. 个股描述类指标

        1. K 线形态：空中楼阁、一针见血、阳克阴、阴克阳

        2. MACD：

            - 背离：顶背离、底背离、

            - 区域：空方区域、多方区域、多方回落0 轴、空方回升 0 轴

            - 转折点：金叉、死叉、即将金叉、即将死叉

            - 警惕点：上升拐头、下降拐头

            - 趋势：上升、下降、走平

        3. 均线：

            - 趋势：多头排列、空头排列，均线缠绕

            - 警惕点：5日线走平，10日线走平，均线走平

            - 转折点：金叉、即将金叉、死叉、即将死叉

        4. DDE

            - 趋势：上升、下降、走平

            - 警惕点：DDX2上升拐头、DDX2下降拐头

        5. 量能

            - 趋势：放量、缩量、平量

            - 区域：爆量区、正常区、地量区
